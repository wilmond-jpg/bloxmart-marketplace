# BloxMart Marketplace

A frontend-only marketplace UI for buying and selling Roblox-style virtual items. The app is a static prototype: all listings, users, and chat data are mocked, and “auth” is simulated with `localStorage`—there is no backend, real OAuth, or payment processing.

## What this project is

BloxMart is a dark-themed marketplace experience with browse/search, item detail pages, seller profiles, fake login/signup, payment method explainers, and a messages UI. It is built for design and UX iteration, not production commerce.

**In scope**

- Browse and filter listings (client-side search and filters)
- Item detail with gallery, seller card, and “Buy Now” → account prompt modal
- User profiles (listings, sales, reviews tabs)
- PHP / USD currency toggle (persisted, live price formatting)
- Simulated auth and messaging

**Out of scope**

- Database, APIs, real auth/OAuth, payment SDKs, admin tools

## Project structure

```
src/
├── routes/           # File-based pages (see src/routes/README.md)
├── components/       # Navbar, Footer, ItemCard, BuyModal, ui/
├── context/          # AuthContext, CurrencyContext
├── data/             # Mock items, users, reviews, messages
├── lib/              # formatters, utils, server helpers
├── assets/           # Item images and static assets
└── styles.css        # Design tokens (brand colors, surfaces)
```

`src/routeTree.gen.ts` is generated by the router plugin—do not edit it manually.

## Conventions for developers

- **Colors:** Use semantic classes from `src/styles.css` (`bg-card`, `text-brand-red`, etc.)—avoid hard-coded hex in components.
- **Auth:** `AuthContext` treats any login/signup submit as success and persists a flag in `localStorage`.
- **Currency:** `CurrencyContext` toggles PHP/USD and formats prices via `src/lib/format.ts`.
- **Mock data:** Extend `src/data/*.ts` for new listings or users; wire images from `src/assets/items/`.
- **Routing:** Add new pages as files under `src/routes/`; layout shell lives in `__root.tsx`.

## Design tokens

Background `#09090b`, surfaces `zinc-900`, borders `zinc-800`, brand red `#E2231A`, brand blue `#00A2FF`, Inter for typography. Cards use `rounded-2xl` with a subtle ring and hover lift.
